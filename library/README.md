# Skill and Agent Library

`optional-skills/` contains useful skills not installed into the active `.agents/skills` context. Move a skill into `.agents/skills/` only when you need it often.

Recommended activation order:

1. product skills: `pm-feedback`, `pm-metrics`, `pm-roadmap`, `pm-user-stories`;
2. output skills: `design-minimal`, `html-draft`, `readme-generator`;
3. operations skills: `meeting-copilot`, `tg-bot-ops`;
4. media skills: `art-director`.
