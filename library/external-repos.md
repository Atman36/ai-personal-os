# External Repositories: What to Use

## The Agency

Use as a specialist catalogue. Extract roles, do not install all agents at once.

Best initial roles:
- Chief of Staff
- Business Strategist
- Product Manager
- Growth Hacker / Head of Growth
- Finance Tracker / CFO
- Software Architect
- Security Architect / Governor equivalent
- Analytics Reporter
- ZK Steward / Memory Steward

## Skills for Real Engineers

Use as engineering discipline when building the OS itself:
- grilling before implementation;
- shared language / context doc;
- TDD;
- diagnose;
- handoff;
- improve architecture.

Install later with the upstream installer if you are actively coding inside this repo.
