# Roadmap

## v0 — Local Codex OS

- Markdown/JSON control plane.
- Custom agents.
- Core skills.
- `osctl.py` validate/sync/status/add-task.

## v1 — GitHub as execution layer

- Use `gh-issues`, `manager`, `task-routing`.
- Create issues from task contracts.
- Link PRs and handoffs.
- Keep local OS as strategy/control layer.

## v2 — Personal memory

- Import selected Obsidian context.
- Create memory index.
- Add context hygiene rules.
- Keep sensitive journal/health data opt-in.

## v3 — Proactive cadence

- Daily/weekly prompts.
- Stale task detection.
- Decision review dates.
- Lightweight metrics dashboard.

## v4 — External tools

- Calendar/task manager.
- Email/Telegram draft workflows.
- Browser/research with citations.
- Approval logs.

## v5 — Agent corporation

- Department repos.
- Agent scorecards.
- Evals per recurring workflow.
- Governor-controlled autonomy expansion.
