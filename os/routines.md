# Routines

## Daily Start — 10 minutes

Prompt: `.codex/prompts/daily-start.md`

Outputs:
- Top 3 actions;
- blocked/stale tasks;
- one decision that needs attention;
- recommended delegation.

## Session Close — 5 minutes

- Update active-work if state changed.
- Write handoff if work is incomplete.
- Capture one lesson or friction point.

## Weekly Operating Review — 45 minutes

Prompt: `.codex/prompts/weekly-review.md`

Outputs:
- wins, misses, blockers;
- tasks to close/defer/continue;
- next week outcomes;
- OS improvement candidates.
