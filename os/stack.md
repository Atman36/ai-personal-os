# Stack

## Current Layer

- Codex as the operating surface.
- Markdown as durable human truth.
- JSON as machine-readable control plane.
- Python scripts as local deterministic tools.
- Skills as reusable workflows.
- Codex custom agents as role-specific workers.

## Future Tool Body

1. GitHub: issues, PRs, repos, specs.
2. Calendar/tasks: scheduling and reminders.
3. Email/Telegram: inbox triage and drafts, not autonomous sends initially.
4. Browser/search: research with citations.
5. Obsidian/files: personal knowledge memory.
6. CRM/finance: only after governance is stable.

## Rule

Add one connector at a time. For every connector define: allowed actions, denied actions, approval gates, audit log, rollback, and eval.
